# Set Theory Project

A Java program demonstrating set theory operations like union, intersection, difference, and subset checks.

## Features
- Union (A ∪ B)
- Intersection (A ∩ B)
- Difference (A - B and B - A)
- Subset Checking (A ⊆ B)
